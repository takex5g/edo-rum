あなたはエディトリアルデザインに強い Web デザイナーです。
目的：罫線（ボーダー）を主役にした、編集紙面のようなミニマル Web サイトのデザインを提案してください。

要件：

- 画面全体をグリッド（例：12 列）で構成し、各コンテンツは必ず「枠（ボーダー）」の中に配置する
- カード UI の区切りはシャドウ禁止。境界線（1px〜2px）だけで分割する
- 余白と整列を厳格に。線がレイアウトの骨格になる
- 配色：白〜薄いグレー基調＋文字はほぼ黒。アクセントは最小（1 色まで）
- タイポグラフィ：見出しはやや太く、本文は読みやすい。装飾は控えめ
- コンテンツ：記事一覧（サムネ＋タイトル＋メタ情報）、ニュース、ショップなどが“面”で並ぶ
- UI の雰囲気：雑誌の目次／台割、フラット、規律、静かな緊張感

出力：

1. 画面構成（ヘッダー / リスト / フッター）のワイヤー説明
2. コンポーネント仕様（カード、タグ、メタ情報、画像比率）
3. ボーダールール（線幅、角、余白、グリッド単位）
4. レスポンシブ（PC/タブレット/スマホでの枠の崩し方）
