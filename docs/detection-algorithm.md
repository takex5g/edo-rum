# 江戸走り検出アルゴリズム

## 概要

本システムは、MediaPipe Pose Landmarkerを使用してリアルタイムに人体の姿勢を検出し、「江戸走り」の特徴的なフォームを判定します。

## 江戸走りとは

江戸走りは、日本の伝統的な走り方で、現代のランニングフォームとは異なる特徴を持ちます：

- **腕の動き**: 左右の腕が前後に対向する（一方が前、一方が後ろ）
- **足の動き**: 左右の足が前後に対向する（一方が前、一方が後ろ）
- **膝の状態**: 膝を曲げた状態で走る

## 検出条件

江戸走りと判定されるには、以下の **3つの条件すべて** を満たす必要があります。

### 条件1: 腕が対向している (armsOpposed)

**判定ロジック**: 手首が肩より「前」にあるか「後ろ」にあるかをX座標で判定

```
1. 体の向きを判定
   facingRight = leftShoulder.x > rightShoulder.x
   （左肩が右肩より右にある → 右向き）

2. 手首と肩のX座標差を計算
   leftArmDiff = leftWrist.x - leftShoulder.x
   rightArmDiff = rightWrist.x - rightShoulder.x

3. 前後を判定
   右向きの場合: 差 > 0 なら「前」
   左向きの場合: 差 < 0 なら「前」

4. 対向判定
   armsOpposed = 左腕が前 XOR 右腕が前
   （一方が前、もう一方が後ろ）
```

### 条件2: 足が対向している (feetOpposed)

**判定ロジック**: 足指が腰より「前」にあるか「後ろ」にあるかをX座標で判定

```
1. 足指と腰のX座標差を計算
   leftFootDiff = leftFootIndex.x - leftHip.x
   rightFootDiff = rightFootIndex.x - rightHip.x

2. 前後を判定（腕と同じロジック）
   右向きの場合: 差 > 0 なら「前」
   左向きの場合: 差 < 0 なら「前」

3. 対向判定
   feetOpposed = 左足が前 XOR 右足が前
```

### 条件3: 膝が曲がっている (kneesBent)

両膝の角度を計算し、閾値未満であるか判定します。

```
leftKneeAngle = angle(leftHip, leftKnee, leftAnkle)
rightKneeAngle = angle(rightHip, rightKnee, rightAnkle)
kneesBent = leftKneeAngle < KNEE_ANGLE_MAX && rightKneeAngle < KNEE_ANGLE_MAX
```

## パラメータ一覧

| パラメータ | 値 | 説明 |
|-----------|-----|------|
| `CROSS_THRESHOLD` | 0.001 | X座標差の閾値（前後判定） |
| `KNEE_ANGLE_MAX` | 165° | 膝が曲がったと判定する最大角度 |
| `MIN_VISIBILITY` | 0.5 | ランドマークの最小可視性（0.0-1.0） |
| `HOLD_MS` | 1000ms | ポーズ保持時間（検出確定に必要な時間） |

## 角度計算アルゴリズム

3点 (A, B, C) から角度 ∠ABC を計算します。

```typescript
function angle(a: Landmark, b: Landmark, c: Landmark): number {
  const ab = { x: a.x - b.x, y: a.y - b.y };
  const cb = { x: c.x - b.x, y: c.y - b.y };

  const dot = ab.x * cb.x + ab.y * cb.y;
  const magAB = Math.hypot(ab.x, ab.y);
  const magCB = Math.hypot(cb.x, cb.y);

  const cosAngle = dot / (magAB * magCB);
  return Math.acos(cosAngle) * (180 / Math.PI);
}
```

## 判定結果の視覚化

検出結果は画面上に以下のように表示されます：

- **膝の角度**: 円弧グラフで現在の角度を表示（165°が閾値）
- **腕の前後**: 肩から手首方向への矢印 + 「外旋」「内旋」ラベル
- **足の前後**: 腰から足指方向への矢印 + 「外旋」「内旋」ラベル
- **色分け**: OK条件は緑色、NG条件は赤色
